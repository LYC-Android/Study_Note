文本格式化
  控制文本
   -指定字体
    -font-family:value,value2;
   -字体大小
    -font-size:value;
   -字体加粗
    -font-weight:normal/bold;
   -文本颜色
    -color:value;
   -文本排列
    -text-align:left/right/centerl
   -文字修饰
    text-decoration:none/underline;
   -行高(文字在其行高范围内垂直居中，当行高=元素高的时候，会导致文字垂直居中 )
    line-height:value(1.6em);
   -首行文本缩进
    text-indent:value(2em);


    body{	
    font-family:'微软雅黑','文泉驿正黑','黑体'；
    }

    h1{
    font-size:40px;
    text-align:center;
    }

表格样式
-表格样式可以使用box模型(边框、内边距、宽、高)以及文本格式化属性

表格特有样式属性
-如果设置了单元格的边框,相邻单元格的边框会单独显示,类似于双线边框
-border-collapse属性:合并相邻的边框
 -设置是否将表格边框合并为单一边框
 -border-collapse:separate/collapse;

.t2{
	border-collapse:collapse;
}

定位
  定位:定义元素框相对于其他正常位置应该出现的位置,或者相对于父元素、另一个元素甚至浏览器窗口本色的位置

   定位的作用
   -解决元素排列的问题
   -使用定位可以将元素摆放到任意位置

   定位的分类
     -默认的定位
        -默认的情况下，块内元素垂直排列，行内元素左右排列，这种默认排列的方式成为流定位

     特殊的定位
       -浮动定位:让块元素左右排列
       -相对定位:可以让元素以自己为目标产生微小的偏移
       -绝对定位:可以让元素以父亲为目标产生很大的偏移
       -固定定位:可以让元素以窗口为目标产生巨大的偏移
       >后三种定位都是以某元素为目标产生偏移 

     浮动定位
      -作用可以让块左右排序
      -分类：左浮动/右浮动
      -浮动的步骤:
       浮动的目标离队
       目标的弟弟跟上
       目标到达指定位置
      -规则:目标脱离流(队伍)

    浏览器渲染的规则
    1.父元素高度会自适应,以其内部流(队伍)中的元素的高度为准，如果元素已经浮了起来，那么就不算是流元素了
    2.浏览器认为文字十分重要,无论如何都会将文字显示完整

    3.1只消除对叔叔的影响
     - 父元素的边框不需要显示
     clear:left;
    3.2消除对父亲的影响
     -父元素的边框要显示
       1.在父元素内加空块
       2.对此空块clear
       3.空块会出现在浮动元素下方
       4.因为空块还在流内，从而拉伸了父元素的高度
